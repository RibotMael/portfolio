La Bataille Java

Une implémentation complète du célèbre jeu de cartes, développée de A à Z en Java avec un affichage interactif via la ligne de commande (Terminal).

Rendu Terminal

Le programme s'exécute dans la console. Il affiche le déroulé de chaque tour, gère les égalités (les fameuses "batailles"), accumule les pots et annonce le vainqueur final.

Architecture POO Stricte

Le projet est construit de manière modulaire autour d'entités claires : Carte (composée d'une Hauteur et d'un Symbole), Tas (deck du joueur) et Joueur.

01. Structure POO

Comme évoqué, le projet repose sur une architecture objet stricte. Chaque élément de la vie réelle correspond à un objet Java.

Classe Carte

La classe Carte n'utilise pas de simples variables textuelles, mais fait appel à des sous-objets Hauteur (ex: Valet, rang 11) et Symbole (ex: Coeur, rouge).

Classe Joueur

Un Joueur possède un nom et son propre Tas (sa main). Il encapsule la logique de jouer une carte.

public class Carte {
    private Hauteur hauteur;
    private Symbole symbole;

    public Carte(Hauteur hauteur, Symbole symbole) {
        this.hauteur = hauteur;
        this.symbole = symbole;
    }

    public Hauteur getHauteur() { return hauteur; }
    public Symbole getSymbole() { return symbole; }

    @Override
    public String toString() {
        return hauteur + " de " + symbole.getNom();
    }
}

02. Logique de Collection

La gestion du paquet de cartes (la classe Tas) utilise une LinkedList issue de java.util. Ce choix est justifié par la nature du jeu (logique de file / FIFO : First In, First Out) : on retire les cartes du dessus et on ajoute les gains en dessous.

  • ✅ pollFirst() : Complexité O(1) pour jouer la carte du dessus.
  • ✅ addLast() : Complexité O(1) pour ranger les cartes gagnées en dessous.
  • ✅ Collections.shuffle() : Utilisé pour le mélange initial du paquet.
import java.util.LinkedList;

public class Tas {
    private LinkedList<Carte> cartes;

    public Tas() {
        cartes = new LinkedList<>();
    }

    public void ajouterCarte(Carte c) {
        cartes.addLast(c); // Ajout en dessous
    }

    public Carte retirerCarte() {
        return cartes.pollFirst(); // Pioche au dessus
    }
    
    public boolean estVide() {
        return cartes.isEmpty();
    }
}
